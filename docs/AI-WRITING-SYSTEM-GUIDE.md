# AI驱动的写作系统 - 完整使用指南

## 🎉 系统功能总览

现在你可以通过AI对话完成几乎所有的项目管理任务！系统已经完全集成，提供了流畅的AI辅助写作体验。

---

## 📚 主要功能模块

### 1. 智能对话系统
**位置**：左侧栏 → AI对话区

**功能**：
- ✅ 与AI进行自然语言对话
- ✅ 智能分析你的意图
- ✅ 自动提供操作建议卡片
- ✅ 直接执行各种操作（创建、修改、删除）
- ✅ 对话历史自动保存
- ✅ 支持多种对话类型（通用、角色、剧情、世界观等）

**使用方式**：
1. 在底部输入框中输入你的需求
2. AI会自动分析你的意图
3. 如果需要执行操作，会显示建议卡片
4. 选择一个建议，点击"执行此操作"
5. 确认后操作会自动执行

---

### 2. 操作卡片系统

**支持的12种操作类型**：

#### 📝 角色管理
1. **CREATE_CHARACTER** - 创建新角色
   - 示例："帮我创建一个叫张三的主角"
   - 支持参数：name, role, appearance, personality, background

2. **UPDATE_CHARACTER** - 更新角色信息
   - 示例："把张三的性格改成更内向"
   - 支持参数：characterId, updates

3. **DELETE_CHARACTER** - 删除角色
   - 示例："删除角色李四"
   - 支持参数：characterId

#### 🌍 世界观设定
4. **CREATE_WORLD_SETTING** - 创建世界观设定
   - 示例："创建一个关于魔法的世界观设定"
   - 支持参数：category, name, description, items

5. **UPDATE_WORLD_SETTING** - 更新世界观设定
   - 示例："更新魔法设定，增加一些新的规则"
   - 支持参数：settingId, updates

6. **DELETE_WORLD_SETTING** - 删除世界观设定
   - 示例："删除关于地理的世界观设定"
   - 支持参数：settingId

#### 📖 章节管理
7. **CREATE_CHAPTER** - 创建新章节
   - 示例："创建第三章：冒险开始"
   - 支持参数：title, order, summary

8. **UPDATE_CHAPTER** - 更新章节内容
   - 示例："把第一章的内容修改一下，让主角更早出场"
   - 支持参数：chapterId, updates

9. **DELETE_CHAPTER** - 删除章节
   - 示例："删除第二章"
   - 支持参数：chapterId

#### 🔗 关系和剧情
10. **ADD_RELATIONSHIP** - 添加角色关系
    - 示例："让张三和李四成为兄妹"
    - 支持参数：character1Id, character2Id, relationship

11. **CREATE_PLOT** - 创建情节线
    - 示例："创建一条关于复仇的情节线"
    - 支持参数：title, description, type

12. **CREATE_OUTLINE** - 创建大纲
    - 示例："为整个故事创建一个三幕式大纲"
    - 支持参数：title, description, structure

### 3. 对话历史管理
**位置**：左侧栏 → 对话列表

**功能**：
- ✅ 创建新的对话线程
- ✅ 查看所有历史对话
- ✅ 按类型筛选对话（通用、角色、剧情、世界观等）
- ✅ 删除对话
- ✅ 清空所有对话
- ✅ 显示最后更新时间

**使用**：
- 点击"+"按钮创建新对话
- 悬停对话可显示删除按钮
- 使用筛选下拉框按类型查看对话

### 4. 键盘快捷键

| 快捷键 | 功能 | 说明 |
|-------|------|------|
| `Ctrl + Enter` | 发送消息 | 快速发送，无需点击按钮 |
| `Ctrl + F` | 搜索消息 | 在当前对话中搜索内容 |
| `Ctrl + K` | 显示快捷键 | 查看所有可用快捷键 |

## 💡 使用技巧

### 1. 高效提问
- ✅ 明确说出你想做什么（创建、修改、删除）
- ✅ 提供足够的细节（角色名、设定描述等）
- ✅ 说明具体要求（性格、关系、情节等）

**好的例子**：
```
创建一个配角叫李四，他是个商人，性格圆滑但内心善良，是主角的好友。
```

**不够好的例子**：

- 系统会显示详细的操作预览

## 🐛 问题排查

### 如果AI没有识别到操作
1. 检查输入是否清晰明确
2. 尝试更具体地描述需求
3. 可以分步骤进行（先创建，再修改）

### 如果操作执行失败
1. 查看错误提示信息
2. 检查网络连接
3. 刷新页面重试
4. 查看浏览器控制台错误

### 如果界面显示异常
1. 尝试切换主题（深色/浅色）
2. 清除浏览器缓存
3. 刷新页面
4. 尝试使用其他浏览器
